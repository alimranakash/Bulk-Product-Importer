Supported Product Types

Bulk Product Importer supports all the main WooCommerce product types you'll need for most stores:

1. Simple Products

These are standard products without any variations. Think of items like books, electronics, or any product that doesn't come in different sizes or colors. Each simple product has one price and one stock quantity.

2. Variable Products

Variable products are parent products that contain multiple variations. For example, a T-shirt that comes in different sizes (S, M, L, XL) or a phone case available in various colors. The parent product defines the attributes, while variations define the specific combinations.

3. Product Variations

Variations are the individual purchasable items within a variable product. Each variation can have its own price, stock, and image. For example, "Blue T-Shirt - Size Large" would be one variation of the "T-Shirt" variable product.

üí° Pro Tip

When importing, always list products in this order: Simple ‚Üí Variable ‚Üí Variations. The plugin automatically sorts them, but organizing your Excel file this way makes it easier to manage.

How It Works

The import process is straightforward and consists of four main steps:

Step 1: Prepare Your Excel File

Create an Excel spreadsheet with your product data. The plugin recognizes standard WooCommerce export columns, making it easy to prepare your data. Include columns for product type, SKU, name, price, categories, and more.

Step 2: Organize Your Images

Place all product images in an "images" folder. Name them to match the references in your Excel file. The plugin will automatically find and attach them to the correct products.

Step 3: Create a ZIP Archive

Combine your Excel file(s) and images folder into a single ZIP file. You can include multiple Excel files if needed ‚Äì the plugin will process them all.

Step 4: Upload and Import

Navigate to WooCommerce ‚Üí Bulk Importer in your WordPress admin. Upload the ZIP file, select your batch size, and click "Start Import". The plugin handles the rest!
