WooCommerce Action Scheduler

Bulk Product Importer leverages WooCommerce's built-in Action Scheduler ‚Äì the same robust system that powers WooCommerce subscriptions, memberships, and background tasks.

What is Action Scheduler?

Action Scheduler is a scalable, traceable job queue for background processing in WordPress. It's included with WooCommerce and provides:

  • Persistent Queue: Jobs survive server restarts
  • Automatic Retry: Failed jobs are retried automatically
  • Logging: Full history of all actions
  • Admin Interface: View and manage jobs in WooCommerce ‚Üí Status ‚Üí Scheduled Actions

How It Works

1. Upload triggers scheduling
   ‚îî‚îÄ‚îÄ Creates "bpi_process_excel_file" actions for each Excel file

2. Each file action runs
   ‚îî‚îÄ‚îÄ Reads Excel file in chunks
   ‚îî‚îÄ‚îÄ Stores products in database
   ‚îî‚îÄ‚îÄ Creates "bpi_process_batch" actions

3. Batch actions run (every 2 seconds)
   ‚îî‚îÄ‚îÄ Imports 25 products (configurable)
   ‚îî‚îÄ‚îÄ Updates progress
   ‚îî‚îÄ‚îÄ Continues until complete

Configuring Batch Sizes

The batch size determines how many products are processed in each background job. Choose based on your server capabilities:

Batch Size Best For Server Requirements
10 products Shared hosting, low-memory servers 128MB RAM, basic CPU
25 products (default) Standard VPS, most hosting 256MB RAM, decent CPU
50 products Good VPS, dedicated servers 512MB RAM, good CPU
100 products Powerful dedicated servers 1GB+ RAM, fast CPU

How to Choose

  1. Start with the default (25 products)
  2. If you see memory errors in logs, reduce batch size
  3. If imports are slow but server is idle, increase batch size
  4. Test with a small import first before large datasets

Real-Time Progress Tracking

During import, you can monitor progress in real-time. The plugin displays:

  • Total Products: Total number of products to import
  • Processed: How many have been processed so far
  • Created: New products added to the store
  • Updated: Existing products modified (same SKU)

Troubleshooting Guide

‚ùå Import Stuck / No Progress

Cause: Action Scheduler not running or cron disabled

Solution:

  1. Check WooCommerce ‚Üí Status ‚Üí Scheduled Actions for pending jobs
  2. Ensure WP-Cron is running: visit your site or use a real cron job
  3. Check error logs for PHP errors

‚ùå Memory Exhausted Error

Cause: Batch size too large for available memory

Solution:

  1. Reduce batch size to 10
  2. Increase PHP memory_limit
  3. Split into smaller Excel files

‚ùå Parent Product Not Found

Cause: Variations listed before their parent variable product

Solution:

  1. Ensure variable products appear before variations in Excel
  2. Verify Parent SKU matches exactly (case-sensitive)
  3. Put variable products in a separate file processed first

‚ùå Images Not Importing

Cause: Image path or filename mismatch

Solution:

  1. Verify images folder is named "images" (lowercase)
  2. Check filenames match exactly (no extra spaces)
  3. Ensure images are valid files (not corrupted)
